## 一、搭建一个express开发目录
1、安装express生成器
`npm i -g express-generator`

2、使用express初始化项目结构
`express --no-view 项目名称`

3、在命令行中指定到项目目录下，并安装依赖
`npm i`

4、启动express服务
```

npm run start
```
